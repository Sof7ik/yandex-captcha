sendFormWrapper(form, sendHandler) → {void}

Отправляет форму form с использованием функции-обработчика sendHandler.

Важно!
После выполнения проверки токена на сервере вызывается

sendHandler(form);

поэтому вам необходимо в вашей функции отправки формы принимать один параметр - DOMElement формы

Parameters:
Name Type Description
form DOMElement

DOM-элемент формы, который необходимо отправить.
sendHandler function

Функция, которая будет вызвана перед отправкой формы. Эта функция должна принимать один параметр: DOMElement формы
